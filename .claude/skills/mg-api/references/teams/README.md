# Microsoft Graph — Teams and Chats Reference

Deep reference for the `mg-api teams` and `mg-api chats` capabilities. The CLI exposes semantic verbs (`mg-api teams list-joined|list-channels|send-channel-message`, `mg-api chats list|messages|send`) that build these endpoints for you. Load this file when you need channel-message body shapes, chat type semantics, or permission notes that the help text does not spell out.
