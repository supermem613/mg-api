# Microsoft Graph — Calendar Reference

The CLI exposes semantic verbs (`mg-api calendar list|view|get|create|update|delete|accept|decline|find-times`) that build these endpoints for you. Load this file when you need event field shapes, time-window semantics, or recurrence details that the help text does not spell out.
