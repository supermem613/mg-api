# Microsoft Graph — Email Reference

Deep reference for the `mg-api email` capability. The CLI exposes semantic verbs (`mg-api email list|get|send|reply|search|move|delete|attachments`) that build these endpoints for you. Load this file when you need OData details, response shapes, or filter syntax that the help text does not spell out.

---

Pick the sibling that matches the task at hand.

- [List and Read Messages](list-and-read.md) — `mg-api email list|get|search|move|delete`, common select fields.
- [Send, Reply, Forward](send-reply-forward.md) — `mg-api email send|reply`, plus Graph-only forward semantics.
- [Attachments](attachments.md) — `mg-api email attachments`, fileAttachment shape, large-file upload session.
- [Mail Folders](folders.md) — inbox, drafts, sentitems, deleteditems, custom folders.
